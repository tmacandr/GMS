# DCW Directory

This directory is a place-holder for the "Digital Chart of the World" (DCW)
map products.

But I can't place my old DCW products here because the copyright still seems
to be active.  It appears that the copyright to the current DCW database is
held by a company:
```
      Environmental System Research Institute (ESRI) Inc
```
Back in the 1990's the DCW product came as a set of CD-ROMs from the
"Defense Mapping Agency" (**DMA**).  Later, they could be purchased from
the "National Imagery and Mapping Agency" (**NIMA**).

For better performance the items on the CD-ROM were copied to the local
disk and the path set in the configuration file.  For GMS, I copied them
to a directory named DCW.  Thus, this place-holder directory.

Then, the path to the DCW products was specified as input configuration file
to the GMS applications.

But now-a-days (2026) I'm sure no one delivers CDs anymore.

I'm not sure if the DCW database comes as a .zip, .tar, etc and/or if it is downloadable.

Also, I'm sure the format of the data-items has changed since then.

But for now, this directory will hold the DCW produts to be processed by
GMS applications.
