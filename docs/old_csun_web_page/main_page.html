A Geographic Map System

Introduction:

The Geographic Map System (GMS) is an MS project. The general objective of this project is to read geographic data from CD-ROMs and then display that data graphically.

The raw geographic data is obtained from:

  • The US Geological Survey (USGS)
  • National Imagery and Mapping Agency (NIMA)
    (formerly the Defense Mapping Agency).
These government agencies generate the map data and store it to CDs, which are then sold to the public.

The following types of maps will be developed:

  • Vector Product Format (aka Digital Chart of the World)
  • Arc Digitized Raster Graphics - Joint Operations Graphic (ADRG-JOG)
  • Digital Terrain Elevation Data (DTED) - Level 0 (only)

Two software toolkits are being developed for this project. The first toolkit is used to read the raw map data (from the CDs). This toolkit is called the GMS I/O Library. The second toolkit provides a series of utilities to allow an application to manipulate the geographic data so it can be graphically displayed. This second toolkit is called the GMS Graphics Library. These graphics utilities assume the application is raster based, but are otherwise independent of any specific graphics platform (e.g. MS Windows GDI vs Motif/Xt/X libraries).
