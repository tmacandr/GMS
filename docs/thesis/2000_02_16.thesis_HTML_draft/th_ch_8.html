Chapter 8 - The GMS ADRG File Toolkit

8.1 - Objectives and Requirements

The toolkit is essentially provides an interface to the data from an ADRG CD-ROM for higher level clients. To achieve this objective the following requirements are specified:

Table 3.1 : Requirements of GMS ADRG File Toolkit
  1. A series of utilities shall be developed to read the data from the files an ADRG CD-ROM.
  2. Access to the following types of files shall be provided:
    • Test Patch Image File
  3. Data structures that correspond to each of the types of files shall be defined to allow client software to process/manipulate the data.
  4. The utilities shall be written in ANSI C.
  5. The utilities shall be compilable by ANSI C and ANSI C++.
  6. The utilities shall be portable to the following platforms:
    • MS Windows 98
    • MS NT (ver xx)
    • HPUX UNIX (Hewlett-Packard)
    • Sun-Solaris UNIX
