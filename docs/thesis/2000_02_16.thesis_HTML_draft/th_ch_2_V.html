2.2) Record Formats of Database Tables

2.2) Database Tables

This document describes the formats of the VPF files that give an overview of the VPF database. These files are called Database Tables.

The files of the VPF database are divided into the following categories:

  • Database Tables
  • Library Tables
  • Primitive Tables
  • Feature Tables
  • Index Tables
  • Value-Description Tables
This document focuses on the database tables. The Database Tables give a general description of the whole VPF database. Although these files are duplicated on each CD, they are not specific to a particular regional library (nor the browse library).

There are two database files (duplicated on each of the 4 CDs). They are:
