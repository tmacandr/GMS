Chapter 6 -
X Window System Using GMS Toolkits

6.1 - Objectives

Applications were developed to demonstrate the portability of the GMS toolkits. These applications were similar in function to the applications developed for Chapter 5. The difference mainly being that the X Window System was used for graphics operations and the Motif toolkit was used to define the user interface.

6.4 - The SOAMAFR Demo

Also, the SOAMAFR-Demo application was reconstructed. The third map, shown in Figure 6.6, shows the portion of the DN theme that contained areas (i.e. polygons) of bodies of water. This information demonstrates lakes and rivers of significant width. In particular, the Niger River and its tributaries are shown as polygons. Also, the very large Niger River Delta is shown. Accuracy can be lost if such bodies of water are shown simply as a line of constant width. library drainage information
