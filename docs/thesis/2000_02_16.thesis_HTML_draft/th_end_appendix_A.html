Appendix A - GMS Utilities

Primitive Data Types in VPF Database

VPF Primitive Datatypes
Data Type Description
T String of n characters
F 32-bit floating point
R 64-bit floating point
S 16-bit integer
I 32-bit integer
C Two dimensional coordinate item (x, y) (two 32-bit floating points)
B Two dimensional coordinate item (x, y) (two 64-bit floating points)
Z Three dimensional coordinate item (x, y, z) (three 32-bit floating points)
Y Three dimensional coordinate item (x, y, z) (three 64-bit floating points)
D Date/Time (20 character string - "YYYYMMDDhhmmss.xxxxx"
X Null field (i.e. 0 bits)
K Triplet Id (one descriptor byte followed by 0 to 12 bytes of data)

    Modifiers:
  • n array of n items
  • * variant length items
  • xxxxx <+> | <-> | hhmm
