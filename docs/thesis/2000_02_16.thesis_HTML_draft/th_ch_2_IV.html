2.1) VPF Record Formats of CoverageTables

1.0 Introduction:

This document describes the formats of the records from Feature Tables of the Vector Product Format (VPF) database.

There are 6 classes of Coverage Tables:

  • Area Feature Tables
  • Line Feature Tables
  • Point Feature Tables
  • Text Feature Tables
  • Value-Description Tables
  • Narrative Tables
